- Линукс - это операционная система с открытм кодом. В основном применяется на серверах и мобильных телефонах (сборка Android).

- CLI - это режим работы в командной строке. Пользователь вводит текстовые команды в терминал и получает ответы тоже в текстовом режиме. Все настройки програм передаются через ключи и параметры команд.
GLI - это графический режим работы. У пользователя есть дталоговые окна, с различными элементами управления - кнопками, флажками. Управлние осуществляется с помощью курсора, котоыре может двигаться мышью. 
Настройка програм осуществляется в основнмо через графические эелменты - кнопки, радиобаттоны, чекбоксы. В случае сервера текстовый режим CLI оказался удобнее, хотя некоторые предпочитают его и для всего остального :)
- SSH (Secure Shell) - это защищенная служба терминального удаленного подключения в текстомовм режиме, как правило к серверу.

- Ключи SSH - генерируются для беспарольного доступа на удаленную машину. Ключей 2 шт: Публичный ключ прописывается (добавляется в определнный файл на сервере) и при логине. Закрытый ключ хранится на машине у владельца.
Генерируются ключи командой "ssh-keygen -t rsa". Ключ - это содержимое файла ключа. По умолчанию публичный ключ хранится в файле id_rsa.pub
Чтобы скопировать свой публичный ключ на сервер нужжно положить его в папку /home/имя пользователя/.ssh.

- df -h - вывести информациюо о дисках, свободном занятом месте. ключ -h - выводить цифры памяти в удобочитаемом виде

- Как найти публичный ключСделать поиск по файлу id_rsa.pub. Обычно ключ лежитв папке /Users/Имя пользователя..ssh  либо  в ~/.ssh/

- gzip - упаковка (Это когда нескольок файлов помещаются в один для удобства пересылки по сети) + сжатие, архивация
