Вопросы

1. **Что такое протокол SSH (зачем нужен)?**
    - Secure Shell (сокр.)
    - Защенный сетевой протокол для подключения к серверу

1. **Какой алгоритм чаще используется для генерации пары ключей public/private?**
    - RSA - криптографический алгоритм ассиметричного шифрования
    - Публичный ключ - (загружаем на сервер/github) - для шифрования
    - Приватный ключ - (держим в тайне) - для расшифровки

1. ********************Зачем подключаться к серверу (напр., с помощью SSH) (типовые задачи)?********************
    1. Диагностика неисправностей
    2. Управление ПО (установка, удаление, настройка)
    3. Развертывание проектов

1. **Какой командой можно посмотреть статистику по дискам на сервере?**
    1. `df -h`

1. ****************************Что такое CLI и для чего используется?****************************
    1. Command Line Interface
    2. Используется для работы на сервере (более практичный)

1. **Как посмотреть свой публичный ключ?**
    
    `cat ~/.ssh/id_rsa.pub` (вывод содержимого в терминал)
    

1. **Как удалить все log-файлы (напр., `error.log`, `success.log`) в тек/каталоге?**
    1. **Лог-файл** - файл с записями о событиях в хронологическом порядке
    2. `rm filename` (для удаления файла)
    3. `rm *.log` (для удаления всех файлов с расширением `.log`)

1. **Что делать, если приватный ключ скомпрометирован?**
    1. Сгенерировать новую пару ключей
    2. `ssh-keygen -t rsa`

1. **Что такое “аптайм” (uptime)?**
    1. Время непрерывной работы сервера
    2. Говорит о стабильности работы

1. **Как посмотреть список подключений к серверу по SSH?**
    
    `w`
    

### Компрессия (сжатие) файлов

- Необх. для уменьшения занимаего места на сервере
    - `gzip -9 filename` (сжатие)
    - `gunzip filename.gz` (декомпрессия)

### Выгрузка файлов из интернета на сервер

`wget URL_полная_ссылка`

`wget -O filename URL_полная_ссылка`

`wget -O name.txt https://gist.githubusercontent.com/khaykov/a6105154becce4c0530da38e723c2330/raw/41ab415ac41c93a198f7da5b47d604956157c5c3/gistfile1.txt`

**Как подключиться к серверу по протоколу SSH?**

`ssh <USER>@<SERVER_ADDR>`

- USER: `ec2-user`
- SERVER_ADR: `linux.telran-edu.de`

**Как посмотреть информацию о содержимом каталога?**

`ls -lah`

### ****************************************************Архивация файлов/каталогов****************************************************

`tar cvf name.tar *.txt` (создать архив)

`c` create

`v` verbose

`f` force

`tar xvf name.tar` (распаковать архив)

`x - extract`

`tar cvfz name.tar.gz *.txt` (упаковать и сжать через GZIP одной командой)

`z - сжатие через GZIP`

### Определение формата файла

`file -s filename`

### Запуск задач (программ) по расписанию

**Типовые задачи**

- Резервное копирование
- Мониторинг сервера
- Генерация SSL-сертификатов

**CRON** - программа-сервис для запуска задач по расписанию в Linux

`crontab -e` (открыть на редактирование мой файл-расписание)

`одна строка == одна задача`

`всего - 6 аргументов`

`1-5 (аргументы): когда`

`6 (аргументов): что`

**минута(1) час(2) день_месяца(3) месяц(4) день_недели(5) command(6)**

`* * * * * /some/command` (каждую минуту)

`30 2 * * * /some/command` (запуск в 2.30 каждые сутки)

`0 4 * * 1-5 /some/command` (запуск в 4.00 по будням)

`0 */2 * * * /some/command` (запуск каждые два часа)

`0 18 * 6-8 6-7 /some/command` (запуск летом по выходным в шесть часов вечера)

`* * * * 1,3,5 /some/command` (запуск каждую минуту пн,ср,пт)

`*/15 * * 1 1,4 /some/command` (запуск в январе каждые 15 минут пн и чт)

### Текстовые редакторы

Чаще всего редактируются файлы конфигураций (настройки каких-либо программ)

`nano` (примитивный)

`nano filename`

`CTRL + S` (сохранить)

`CTRL + X` (выйти)

`vim` (продвинутый)

`vim filename`

`:set nu!` (включить нумерацию строк)

`:wq!` (сохранить и выйти)

`:q!` (выйти без сохранения)
